Bonjour. J'utilise PHP et Twig pour mon lab. J'utilise un serveur PHP et
j'envoie un formulaire avec les informations Prenom, Nom et Date. J'utilise la
methode post. Avec les elements, je render une section twig avec les elements
recus. Ensuite, j'opere sur ces elements pour afficher le bon horoscope.

Une fois le serveur lance, vous pouvez aller sur
http://localhost:80 ou http://127.0.0.1:80

J'utilise Twig car il a ete recommande. Il s'agit d'une sorte de Jinja2 et je
m'en sert ainsi pour recuperer mes parametres.

Pour savoir quoi installer et comment, voir le fichier requirements.txt.
